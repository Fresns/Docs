# 互动配置 InteractiveHelper

`App\Helpers\InteractiveHelper`

## 获取用户交互配置

```php
InteractiveHelper::fresnsUserInteractive($langTag);
```
| 参数名 | 类型 | 是否必传 | 说明 |
| --- | --- | --- | --- |
| langTag | String | NO | 语言标签，不传则使用默认语言 |
